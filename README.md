 Amazon E-commerce Analytics Project
 
 Project Overview:

This project analyzes Amazon e-commerce sales data to uncover insights related to:

Revenue performance

Profitability

Customer segmentation (RFM)

Customer Lifetime Value (CLV)

Revenue contribution by segment

The project demonstrates a complete end-to-end analytics pipeline using Python, SQL, and Power BI.

 Business Objective

The goal is to build an interactive dashboard that helps:

Monitor sales trends

Identify top customers

Segment customers using RFM scoring

Analyze Customer Lifetime Value

Support data-driven decision making

 Tech Stack

Python (Pandas, SQLAlchemy)

SQLite

SQL

Power BI Desktop

DAX
